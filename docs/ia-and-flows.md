# 정보 구조 및 화면 흐름

## 사용자 여정
1. **홈/카테고리/검색**에서 상품 탐색
2. **상품 리스트**에서 정렬 및 필터 → 상품 클릭
3. **상품 상세**에서 옵션·수량 선택 → 장바구니 추가 또는 바로 구매
4. **장바구니** 검토 → 주문서로 이동
5. **주문 정보** 입력(주소/결제/할인) → PG 결제 진행
6. **결제 결과** 페이지 확인(완료/실패/입금대기)
7. **마이페이지**에서 주문/배송 확인, 취소/교환/환불 신청

## 주요 화면 섹션
- **상품 리스트**: 그리드/리스트 토글, 정렬(최신/오래된/판매량), 가격·카테고리·옵션 필터, 페이지네이션
- **상품 상세**: 메인 이미지, 썸네일, 가격/할인/적립, 배송 정보, 옵션 선택, 수량 조절, 장바구니/구매 버튼, 리뷰/QA 탭(후속)
- **장바구니**: 항목별 옵션/수량 수정, 삭제, 선택 결제, 예상 결제 금액/배송비/할인 미리보기
- **주문 정보**: 배송지 선택/입력, 연락처, 배송 메모, 쿠폰·포인트 적용, 결제 수단 선택, 약관 동의
- **결제 결과**: 완료(영수증/주문번호), 실패(재시도 안내), 입금대기(가상계좌/무통장) 시 입금 정보
- **주문 내역/상세**: 주문 상태 타임라인, 송장/배송사, 취소/교환/반품 버튼, 환불 정보, 상담 링크

## 관리자 콘솔 레이아웃 가이드
- 좌측 사이드바: 대시보드, 상품, 주문, 프로모션, 회원, 설정 메뉴
- 상단 바: 검색, 알림, 사용자 메뉴(로그아웃)
- 콘텐츠: 카드형 지표 위젯, 테이블/폼은 부트스트랩 컴포넌트 기반으로 일관성 유지

## 관리자 주요 흐름
- **상품 등록**: 기본 정보 → 이미지/옵션 → 재고/가격 → 공개 여부 → 저장
- **주문 처리**: 주문 검색/필터 → 상세 → 송장 등록 → 배송 상태 업데이트 → 취소/교환 처리
- **프로모션 관리**: 쿠폰 생성(타입/값/적용 범위) → 배포 → 사용 현황 확인
- **정산 지원**: 결제/환불 로그 확인, 기간별 매출·수수료 엑셀 다운로드

## 시스템 흐름(요약)
- **인증**: SNS OAuth 로그인 후 액세스 토큰/세션 발급, 신규 사용자는 회원가입 프로필 입력
- **주문/결제**: 장바구니 스냅샷으로 주문 생성 → PG 결제 요청 → 콜백/웹훅으로 결제 상태 동기화 → 주문 상태 업데이트
- **취소/교환/환불**: 주문 상태와 재고/정책 검증 후 PG 취소 또는 회수 프로세스 → 환불 처리 → 상태 로그 기록
